## 研究方向概述

本方向聚焦**边缘约束下的多智能体系统**，将大语言模型的 Agent 能力与实际边缘场景（隐私、离线、算力受限）相结合。

### 3.1 基于多代理 LLM 的边缘模型定制

构建多 Agent 协作框架，实现边缘侧模型的自动适配、评测与持续优化闭环。

**关键技术：** Agent 通信协议 · 角色分工 · 边缘 Agent 编排 · RAG 知识库

### 3.2 多智能体协同决策

面向多无人机/车编队场景，研究分布式协同任务分配、冲突消解与动态规划。

**关键技术：** 强化学习 · 博弈论 · 分布式优化 · 多无人机任务分配

### 3.3 科研助手与工具

面向学术场景的智能助手，支持论文检索与阅读、实验记录与代码评测的自动化。

**关键技术：** 检索增强生成（RAG） · 代码自动评测 · 知识库构建

## 支撑平台

**EdgeAgentWorks** 边缘多智能体应用平台，包含 Agent 编排与工具插件、RAG 知识库、边缘模型定制与权限审计四大模块。
